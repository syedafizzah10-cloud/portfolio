# Fizzah Ehtisham — Portfolio

A premium real estate content portfolio site built with Vite (vanilla JS + CSS).

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and click **Add New Project**
3. Import your GitHub repository
4. Vercel will auto-detect Vite — leave all settings as default
5. Click **Deploy**

Your site will be live at `https://your-project.vercel.app`

## Project Structure

```
fizzah-portfolio/
├── index.html          # Main HTML with all sections
├── src/
│   ├── style.css       # Full CSS (Charcoal + Gold theme)
│   └── main.js         # All interactive JS
├── public/

```
